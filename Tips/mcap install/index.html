    Mcap install

    環境¶

    • ubuntu22.04

    • ros2 humble

    • githubのmcapリポジトリのreleasesで、mcap-cliで最新versionをDLしてくる． Releases · foxglove/mcap

    • chmod +x ./mcap-linux-amd64
    • sudo mv mcap-linux-amd64 /usr/bin/mcap

    using rosbag-convert¶

    そのままやると、topicに数字が入っているせい(topic名のみの問題)でmcapだとros2 bag run <input.bag>をしたときに失敗する．

    • そのまま変換したときのコマンド
      • rosbags-convert --src section.bag --dst rosbags-converted-mcap.mcap --dst-storage mcap
    • 対応策
      • 数字が入っているトピックは重要ではなかったので、必要なトピックをpubして、それをsubscribeした．
